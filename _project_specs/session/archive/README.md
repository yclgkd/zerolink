# Archive

Historical `_project_specs` material lives here.

Rules:
- Keep archive files in reverse chronological order unless a file explicitly says otherwise.
- `completed.md` is the canonical completed-work archive.
- Dated session summaries may also be added here as `YYYY-MM-DD[-topic].md` when a session or feature needs a fuller handoff record.
- Live project state does not belong here; keep that in `../current-state.md`, `../decisions.md`, and `../code-landmarks.md`.
